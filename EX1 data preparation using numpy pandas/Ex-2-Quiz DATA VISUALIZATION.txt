1. What is Matplotlib used for?

A. Data storage
B. Data manipulation and analysis
C. Data visualization
D. Data retrieval
ANSWER: C

3. Which library is built on top of Matplotlib and provides a high-level interface for visualization?
A. Matplotlib
B. Seaborn
C. Both
D. None
ANSWER: B

4. What is the syntax to create a basic line plot in Matplotlib?
A. plt.plot(x, y)
B. plt.line_plot(x, y)
C. plt.graph(x, y)
D. plt.lines(x, y)
ANSWER: A

What is the syntax to add a legend to a line graph plotted using the plt.plot() function?
a. plt.plot(x, y, label='Line 1')
b. plt.plot(x, y, l='Line 1')
c. plt.plot(x, y, legend='Line 1')
