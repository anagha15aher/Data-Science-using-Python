1. What is the main difference between Numpy and Pandas?
A. Numpy is a library for numerical computing
B. Pandas is a library for numerical computing.
C. Numpy is a library for data manipulation and analysis.

5. What is the difference between df.dropna() and df.fillna() in Pandas?
A. df.dropna() removes missing values while df.fillna() adds missing values.
B. df.dropna() adds missing values while df.fillna() removes missing values.
C. df.dropna() removes all rows with missing values while df.fillna() fills missing values with a specified value or method.
D. df.dropna() fills missing values with a specified value or method while df.fillna() removes all rows with missing values.
